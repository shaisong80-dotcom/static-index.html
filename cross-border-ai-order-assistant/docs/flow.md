```mermaid
flowchart TD
    A["ERP/平台订单导出"] --> B["运营上传 CSV 订单"]

    C --> D["异常规则识别"]
    D --> E{"是否存在异常"}

    I --> J["输出原因、动作、话术"]
    J --> K["分派运营/物流/财务/客服处理"]
    K --> L["处理结果复盘与规则优化"]
```
