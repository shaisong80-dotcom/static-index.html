# 简历项目描述

## 推荐写法

**跨境电商订单异常识别与 AI 自动化处理系统**

项目背景：针对跨境电商订单履约中物流单号缺失、地址异常、SKU 缺失、报关金额异常、订单状态不一致等重复人工排查场景，设计并实现 AI 自动化辅助处理工具，提升运营排查效率。

主要工作：

1. 梳理跨境电商订单履约流程，输出 PRD、业务流程图和功能设计方案。
2. 设计订单异常识别规则，覆盖物流、地址、联系方式、SKU、金额、报关信息和状态一致性等场景。
3. 使用 Python 实现订单 CSV 解析、异常检测、优先级判断和统计分析。
4. 搭建 Web 可视化看板，展示订单总数、异常订单数、异常率、异常类型分布和订单明细。
5. 设计 AI 处理建议模块，自动生成异常原因、处理动作、建议责任人和客服/运营沟通话术。
6. 使用 Cursor 辅助完成页面搭建、接口联调和测试优化，验证 AI 自动化在订单处理场景中的可行性。

项目成果：

- 支持 9 类高频订单异常识别。
- 支持异常订单筛选、优先级标记和 AI 建议生成。
- 完成从需求分析、PRD 输出、流程设计到 Demo 落地的完整产品实践。

## 面试一句话介绍

这是一个模拟跨境电商 ERP 订单同步后的异常处理工具。我先梳理订单履约流程和人工排查痛点，再把物流、地址、SKU、金额、报关和状态异常标准化成识别规则，最后做成 Web 看板，并通过 AI 建议模块生成处理动作和客服话术。
